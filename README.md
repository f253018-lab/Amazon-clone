Amazon Clone 

A front-end clone of the Amazon homepage built using HTML and CSS. This project was created to practice web page structure, CSS styling, Flexbox, and responsive design.

Features:

- Responsive navigation bar
- Search bar
- Hero section
- Product cards
- Footer
- Clean and modern layout

Technologies Used:

- HTML
- CSS
